# KinopoiskApp

## _Приложение для поиска информации о художественных фильмах_

## Описание приложения:
* Приложение состоит из двух экранов: экран поиска и экран детальной информации о фильме;
* Архитектурный паттерн: MVP(Model/View/Presenter);
* Верстка: SnapKit.
